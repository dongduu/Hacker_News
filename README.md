# 해커뉴스

해커뉴스(https://news.ycombinator.com/) 리딩 페이지 만들기

## Stacks
- React
- styled-component
- Axios
- swiper
- figma

## Skill
- 메인 페이지에 Top News 5 표시
- 카테고리별로 뉴스 보기
- 각 카테고리별 뉴스를 최신순/인기순으로 분류 가능
- 디테일 카드와 기본 카드로 분류 가능
- 뉴스 클릭 시 해당 링크로 이동

## Points
- swiper로 슬라이드 구현
- figma를 사용하여 디자이너와 협업
- CRA를 사용하여 SPA 구현
- styled-component 사용
